# Project-Daedalus
## ¿Qué es Daedalus?
Es un Proyecto de Desarrollo de Software con un enfoque académico y comercial
## ¿Qué pretende hacer?
Crear una plataforma multipropósito en donde los usuarios encuentren un amplio repertorio de herramientas de software para el desarrollo de sus actividades, independientemente del tipo que sean Daedalus pretende tener una herramienta para ayudar al usuario. Ver [Project Charter](docs/ProjectCharter.md)
## ¿Cómo está organizado?
Existe un núcleo central organizador y distribuidor, en él se encuentran todas los módulos seleccionables según los requerimientos de cada usuario. Este núcleo es la raiz de la estructura gerárquica y es el maestro de orquesta de todo lo demás en el sistema. Ver [Arquitectura del Proyecto](docs/Architecture)
