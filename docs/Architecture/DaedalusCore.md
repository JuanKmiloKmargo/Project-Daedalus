# Daedalus Core (Núcleo)
## ¿Qué hace?
- Organiza el funcionamiento de todo el sistema, su funcionamiento es central e independiente de todas las entidades que aloja.
- Controla la comunicación entre módulos según las características propias del módulo.
- Gestiona la información y las preferencias del Usuario
- Mantiene segura la inforación del Usuario (Implementación Futura)
- Verifica la adquicisión de licencias de pago (Implementación Futura)
## ¿Qué aloja?
- Módulos Funcionales
   - Submódulos
- IA Icarus (Implementación futura)
- Canal de comunicación entre Módulos
