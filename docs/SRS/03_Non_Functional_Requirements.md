# Requisitos No funcionales
## Rendimiento
Una búsqueda deberá completarse en menos de un segundo con decenas de miles de productos.
## Escalabilidad
El sistema deberá soportar múltiples sucursales.
## Modularidad
Cada módulo deberá instalarse sin modificar el Core.
## Seguridad
Los Usuarios solo podrán acceder a Módulos autorizados
## Disponibilidad
El sistema deberá poder funcionar incluso cuando algunos módulos estén deshabilitados.
## Compatibilidad
El backend deberá ser independiente de la interfaz.
## Usabilidad
Las operaciones frecuentes deberán poder realizarse con el menor número posible de acciones.
## Mantenibilidad
Toda funcionalidad deberá estar documentada.
## Internacionalización
El sistema deberá admitir múltiples idiomas.
## Accesibilidad
La interfaz deberá ser utilizable por personas con diferentes capacidades.
## Auditoría
Toda modificación importante deberá quedar registrada.
## Portabilidad
El Core debe poder ejecutarse en distintos sistemas operativos o en contenerdores
